# **Chan Wars 2: Electric Boogaloo**
Fight your way through the bosses with this unique memory game, loosely based on Westmount C.I. teachers!

Dependencies
* Python 3.10.X and up
* Pygame
* PyYAML

Bosses
1. Devil Chan
2. Ms. G
3. Mr. Phone
